# DeathNote

A simple note-taking app with a Death Note theme, built with Python and NiceGUI.

## Features

- Write and save notes
- View saved notes from `death_notes.json`
- Delete notes with confirmation
- Marks a note as "DIED" after 40 seconds

## Requirements

- Python 3
- NiceGUI

Install NiceGUI:

```bash
pip install nicegui
```

## Run

```bash
python main.py
```

Open the app in your browser:

```text
http://localhost:8081
```

You can change the port with the `PORT` environment variable.
